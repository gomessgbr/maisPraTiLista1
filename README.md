# Lista 1

Essa lista tem como intuito praticar lógica de programação


## Como exercutar um exercício

- Clone o repositório 
  ```bash
    git clone https://github.com/gomessgbr/maisPraTiLista1.git
  ```
- Instale as dependecias
  ```bash 
    npm install
  ```
- Execute o exercício desejado 
  ```bash
    node exercico1.js
  ```
